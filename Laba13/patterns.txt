Builder -Использован для сборки уведомления
Singleton - Использован для создания единственного экземпляра базы данных
Factory Method - Использован для создания экземпляров ViewModel, так как они требуют аргументы
Adapter - Использован для реалзации RecycleView для отображения игр и реализации взаимодействия с ними
Decorator - Использован для создания экземпляров модели бд, а также ленивого средства доступа к ней
